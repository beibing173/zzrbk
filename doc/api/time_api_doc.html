time_api.php API文档

1. API概述

`time_api.php` 是一个简单的API接口，用于获取当前服务器时间。该接口返回JSON格式的数据，包含当前时间。

2. 请求方法

- HTTP方法: GET
- URL: https://zzrbk{$rthSuffix}/API/time_api.php

3. 请求参数

该API接口不需要任何请求参数。

4. 返回值

返回值是一个JSON对象，包含以下字段：

字段 类型 描述
success 布尔值 表示请求是否成功。
time 字符串 表示当前服务器时间，格式为 YYYY-MM-DD HH:MM:SS。

5. 示例

5.1 请求示例

使用curl命令调用API：

curl -X GET https://zzrbk{$rthSuffix}/API/time_api.php

5.2 返回示例

成功返回：

{
    "success": true,
    "time": "2025-01-27 23:46:45"
}

失败返回：

{
    "success": false,
    "message": "无法获取时间"
}

6. 错误处理

如果请求失败，API将返回一个包含错误信息的JSON对象。

7. 关于

服务器时间=北京时间（UTC+8）

原理来自时钟:https://zzrbk{$rthSuffix}/gongju/time.html
